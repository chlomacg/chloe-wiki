Brevis: About

Brevis is a concise and portable impure functional programming language, developed by Chloe MacGregor. The compiler for it is currently written in Rust, and itself can compile Brevis to both C and JavaScript. Its development, sparked by frustration with the C programming language, began in May of 2021. The compiler's source is hosted on Sourcehut.
